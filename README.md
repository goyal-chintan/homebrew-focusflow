# homebrew-focusflow

[Homebrew](https://brew.sh) tap for [FocusFlow](https://github.com/goyal-chintan/focus-flow) — a native macOS menu bar focus timer with on-device AI coaching.

## Install

```sh
brew tap goyal-chintan/focusflow
brew install --cask focusflow
```

## Requirements

macOS 26 (Tahoe) or later.

## About FocusFlow

FocusFlow is a Pomodoro-style focus timer that lives in your menu bar. It includes an on-device focus coach, app and website tracking, session analytics, and system-level distraction blocking.
